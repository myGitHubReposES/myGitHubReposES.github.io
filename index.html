Pruebas de compatibilidad entre navegadores de dispositivos móviles

Descripción

Esta página contiene diferentes elementos input file donde se puede ver como se puede configurar el atributo accept y capture.

Atributo accept

Formatos válidos: JPG, PNG, BMP, TIFF y PDF

    
        <input type="file" accept="image/*,.pdf" />

Atributo capture

Especifica que, opcionalmente, se debe capturar un nuevo archivo y qué dispositivo se debe utilizar para capturar ese nuevo medio (user, camera o environment).

    
        <input type="file" capture />

Lista de Navegadores

  • Safari 17 a 14
  • Chrome, Firefox Y edge

Elementos de testeo

Input sin atributos

Input con atributo accept = "image/*"

Input con atributo accept = "image/*, .pdf"

Input con atributo accept = ".jpg, .png, .bmp, .tiff, .pdf"

Input con atributo capture

Input con atributo accept = "image/*" y ;capture=camera
